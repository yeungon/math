# math-generator

> My awesome module

Inspirated by https://github.com/lukew3/mathgenerator, https://www.mathgoodies.com/worksheets/generator and https://mathsbot.com/

## Install

```sh
npm install math-generator
```

## Usage

```js
import {random} from 'math-generator';

random(10);
//=> '[1 - > 9]'
```
